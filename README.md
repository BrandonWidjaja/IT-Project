### Prerequisites:
1. Node.js (v17.8.0)
2. React (18.2.0)
3. Mongoose (6.5.4)

### Start Guide:
#### Frontend:
1. 
```
cd client
```
2. 
```
npm start
```
3.  open browser (http://localhost:3000)

#### Backend:
1. root directory:
```
npm start
```
